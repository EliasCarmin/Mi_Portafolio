Predicción Copa América

El presente proyecto de Data Science tiene como objetivo pronósticar el futuro campeón de la Copa América 2024 utilizando técnicas de web scraping para la recolección de datos y procesos ETL para el tratamiento de los mismos, para el pronóstico se utilizó un algoritmo de probabilidad aunque en un futuro se pretende reemplazarlo por un algoritmo de Machine Learning

Tecnologías Usadas

  • PYTHON
  • PANDAS
  • Jupyter NoteBook
  • Pycharm/Anaconda

Características

  • Algoritmo de probabilidad de Poisson
  • Se utilizó Anaconda que es una distribución de software libre y de código abierto de los lenguajes Python y R, ​ utilizada en ciencia de datos y aprendizaje automático.
  • Técnicas de web scraping y análisis de datos utilizadas.

Retos y Experiencias Afrontados

Los desafíos afrontados fueron nuevos y sastifactoriamente afrontados:

1. Este proyecto es el primero en donde he hecho uso del lenguaje Python

2. Para la recopilación de los datos de entrenamiento usé conocimientos de web scraping para la extracción de las tablas al IDE

3. Al momento de la recopilación hubo dificultades en los formatos HTML de las páginas web, por ello se utilizó la librería Selenium

4. Para la transformación y limpieza de los datos se usó la librería Pandas
