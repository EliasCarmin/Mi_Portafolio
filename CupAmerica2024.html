Predicción Copa América

View Project Code in Git Hub

El presente proyecto de Data Science tiene como objetivo pronósticar el futuro campeón de la Copa América 2024 utilizando técnicas de web scraping para la recolección de datos y procesos ETL para el tratamiento de los mismos, para el pronóstico se utilizó un algoritmo de probabilidad aunque en un futuro se pretende reemplazarlo por un algoritmo de Machine Learning

Tecnologías Usadas

  • PYTHON
  • PANDAS
  • Jupyter NoteBook
  • Pycharm/Anaconda
