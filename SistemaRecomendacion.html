Sistema de Recomendación

View Dashboard Power Bi

El presente proyecto de Sistema de Recomendación tiene como objetivo realizar un proceso de recomendación de un producto para un cliente B basado en un cliente A, utilizando el algoritmo de similitud de coseno, que pertenece al filtrado colaborativo. El dataset del estudio fue obtenido de un repositorio público de Microsoft. Utilizando SQL Server, se realiza la consulta y, con Python y sus respectivas librerías, se llevan a cabo los procesos ETL. Finalmente, se diseña el dashboard con Power BI.

Tecnologías Usadas

  • PYTHON
  • PANDAS
  • ScikitLearn
  • Power Bi
  • SQL Server

Proyecto Screenshots

Características

  • Dashboard interactivo para el mejor entendimiento del usuario.
  • Se utilizó filtrado colaborativo con el algoritmo de similitud del coseno.
  • El dashboard de Power Bi está conectado con la base de datos SQL Server ejecutando un script de Python.

Retos y Experiencias Afrontados

Los desafíos afrontados fueron nuevos y sastifactoriamente afrontados

1. Este proyecto me permitió interactuar diferentes herramientas en diferentes entornos para un único fin

2. Los datos que se utilizaron del estudio se obtuvo en un dataset público de microsoft

3. Se procuró tener una paleta de colores familiar para el diseño de los dashboards

4. Para la transformación y limpieza de los datos se usó la librería Pandas y la interfaz de transformación de datos en Power Bi
