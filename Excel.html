Macros y VBA en Excel

los Siguientes proyectos son macros avanzados siendo programados utilizando VBA(Visual Basic For Aplications), implementando la transferencia de datos de excel a la nube, uniendo la lógica del negocio junto con interfaces rápida e intuitivas.

Conocimientos Empleados

    1. Fundamentos de VBA

    2. Estructuras de Control

    3. Manejo de Objetos en Excel

    4. Uso de Funciones y Procedimientos

    5. Manejo de Eventos

    6. Manejo de Formularios y Controles (UserForms)

    7. Automatización y Manipulación de Archivos

Retos y Experiencias Afrontados

Los desafíos afrontados:

1. Problema del rendimiento al manejar miles de registros provocando que la app se vuelva lenta

2. Manejo de la relación de las tablas

3. Crear automatizaciones para movimientos de datos entre tablas, generar reportes y ejecutar procesos internos.

4. Personalización de la interfaz limitada, no se puede modificar el diseño como una app nativa

5. Seguridad y control de usuarios para roles con permisos

6. Crear automatizaciones para movimientos de datos entre tablas, generar reportes y ejecutar procesos internos.
